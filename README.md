## Deskripsi
Jelaskan secara singkat di sini mengenai aplikasi yang Anda bangun menggunakan PHP Native dan kombinasi frontend HTML, CSS, serta JavaScript. Aplikasi ini dirancang dengan fokus pada kesederhanaan dan fungsionalitas yang efisien.

## Fitur Utama
* **Pure PHP:** Dibangun tanpa framework untuk performa yang ringan.
* **Responsive UI:** Tampilan yang menyesuaikan perangkat menggunakan CSS murni.
* **Interaktivitas:** Validasi dan manipulasi DOM menggunakan Vanilla JavaScript.
* **Manajemen Database:** Integrasi database menggunakan MySQLi atau PDO.
